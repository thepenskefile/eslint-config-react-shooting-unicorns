# shooting-unicorns-eslint-config

## Install

```
$ npm install @shooting-unicorns/shooting-unicorns-eslint-config --save-dev
```


## Usage

In `.eslintrc` or `.eslintrc.json` add:

```
{
  "extends": [
    ...
    "@shooting-unicorns/shooting-unicorns-eslint-config",
    ...
  ],
  ...
}
```
